HTML Liste as principais tags HTML e descreva suas propriedades e utilizações. Inclua exemplos práticos. • Estruturais: A estrutura HTML é a forma como o conteúdo de uma página web é organizado e estruturado. É uma tag que abre um documento HTML. Depois de abrir a tag , é necessário fechá-la no final do documento. As tags HTML são elementos que informam ao navegador como formatar o texto e definir a estrutura e o significado do conteúdo de uma página. Elas são marcadas por sinais de “maior que” e “menor que”. É uma coleção de metadados, ou seja, dados sobre os dados, que fornece informações gerais sobre um documento. A tag é utilizada para passar informações ao navegador que não serão apresentadas na tela para o usuário. Algumas informações que podem ser incluídas na tag são: Título do documento, Links para scripts e folhas de estilos. É responsável por conter o conteúdo visível de um documento, ou seja, todo o corpo do site que é exibido na janela do navegador.
